# 🧮 Modern Web Calculator

A sleek, interactive, and dark-themed calculator built from scratch using core web technologies. This project handles standard arithmetic operations, provides real-time result updates, and includes full keyboard support for a great user experience.

## ✨ Features

- **Standard Operations:** Addition (`+`), Subtraction (`-`), Multiplication (`×`), and Division (`÷`).
- **Extra Functions:** Percentage (`%`) and positive/negative sign toggle (`+/-`).
- **Keyboard Friendly:** You can use your laptop/PC keyboard directly (Numpad, Enter, Esc, Backspace).
- **Beautiful UI:** A modern dark-mode aesthetic with interactive hover and click effects using CSS Grid.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, Flexbox, and CSS Grid layout.
- **JavaScript:** Vanilla JS for state management and mathematical logic.
